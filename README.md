# ui自动化测试  
## 自动化测试框架目标

* `数据驱动`
* `分布式并发测试`
* `步级报告`
* `步级日志`
* `添加截图至测试报告`
* `封装技巧`
* `失败重试`
* `PageObjects 设计模式`
* `CI 集成`
* `并行 API 接口测试`
* `分布式并行 APP UI 测试`
* 输出 [企业级测试报告](https://andyzrp.github.io/report/report.html/)。
